Bienvenidos a los ejercicios

Aprendices en este espacio realizaremos las pruebas a traves de GitHub, y desarrollaremos las estructuras iniciales.

Primeros pasos

Para realizar los ejercicios, se debe realizar un fork del repositorio, y luego clonar el repositorio en su maquina local.

Una vez clonado el repositorio, se debe crear una rama con su nombre y apellido en minuscula, ejemplo: juanfercalderon y trabajar en ella.

Una vez terminado el ejercicio, se debe realizar un commit y un push a la rama creada.

Finalmente, se debe realizar un pull request para que el profesor pueda revisar el ejercicio.
